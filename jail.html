jail problems

I was recently inspired by the jailctf competition to try a hand at making my own jail-type problems in python. Here are two of my attempts :)

Cold jail

This one should work in any python version.
cold.py 
# like any jail, you need to get the contents of flag.txt
import re

def curry(a):
    def X(b):
        def Y(c):
            # todo: fix typo
            return a(c(b))
        return Y
    return X


code = input()

# it's cold outside, all your functions need something warm to wear
assert re.match(r"^(\([a-z]+\))*$", code), "wrong format"
assert all(x not in code for x in ("eval", "exec", "breakpoint")), "no rce please"
assert len(code) < 105, "too long"

eval(code)

This is definitely the simpler one, but I just thought this way of writing code was funny.

Login

login.py 
import codecs
password = "lucky!"

while password:
    func = [codecs.decode, codecs.encode][input("which one [e/d] ") == "e"]
    password = func(password, input("how "))
    print(password)

print("Welcome to macOS")
print(open("flag.txt").read())

To log into this server and access the file, all you need to do is login with an empty password. Thankfully, all python codecs are reversible, right? I tested my solution on 3.12.
